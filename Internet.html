Topic: 01/05 - Intro to Nets Date: June 7, 2026

THE INTERNET: "Network of Networks"

★ DEFINITION:

The Internet is the global network of computers connected to each other which communicate through a standardized set of protocols (TCP/IP).

Exam MCQ Key!

Vint Cerf is widely recognized as the "Father of the Internet".

It compromises of private, public, academic, business, and government networks spanning local to global scopes, all linked using electronic, wireless, and optical technologies.

🔑 Key Services Carried:
  • 👉 WWW (World Wide Web): Interlinked hypertext docs.
  • 👉 Electronic Mail (E-mail)
  • 👉 Discussion forums & Chat services
  • 👉 Internet Telephony (VoIP) & Media Sharing
💡 Visualizing Data Transfer:

Information moves across the globe in tiny chunks called bits over physical mediums!

🕰️ HISTORICAL ORIGINS:

Born out of research in 1970s commissioned by DARPA (Defense Advanced Research Projects Agency, USA). Early prototype: ARPANET which pioneered packet switching technology.

💡 Quick Memory Trick: "Vint Cerf" sounds like "Surfing" the web. He is the Surf Father!

Topic: 02/05 - Protocols & Security Date: June 7, 2026

PROTOCOLS, ROUTING & SECURITY

🌐 1. IP Addresses & DNS (The Phonebook):
  • • IP Address (Internet Protocol): A unique numerical label assigned to every device on a network. (e.g., IPv4 or IPv6)
  • • DNS (Domain Name System): Acts as the Internet's phone book. It converts human-friendly domain names (like google.com) to machine IP Addresses.
Simple DNS Query Loop:
📦 2. Packets, Routing, and Reliability:

Data is broken down into small blocks called Packets. They do NOT travel on a fixed line. Instead, routers direct each packet along the fastest, most dynamic paths available. If a path is blocked, packets route dynamically elsewhere.

🌐 3. HTTP & HTML:

HTTP (Hypertext Transfer Protocol) is the ruleset for fetching web files. HTML dictates how they look on your screen.

🔒 4. Cryptography & Cybersecurity:

To block intruders, communication is secured using Cryptography. Major layer: SSL/TLS which displays as HTTPS (padlock icon in browser). Uses public and private keys to guarantee secure, encrypted transactions.
